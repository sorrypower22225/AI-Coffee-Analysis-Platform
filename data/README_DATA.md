# Data 說明

本專案保留兩份正式資料與一份部署展示用小資料。

| 檔案 | 用途 |
|---|---|
| `Coffee Shop Sales_RAW.csv` | 原始交易資料，用於重新清洗、資料驗證、基礎分析 |
| `Coffee_Shop_Enterprise_Features_Corrected.csv` | 清洗與特徵工程後資料，用於 Dashboard、AI 模型、Power BI 分析 |
| `Coffee_Shop_Demo.csv` | 小型展示資料，用於 GitHub / Render 測試部署 |
| `location_data.csv` | 展店分析用的門市/候選地點資料 |

已刪除重複檔案：

- `Coffee Shop Sales.csv`
- `Coffee Shop Sales_RAW_BACKUP.csv`

刪除原因：內容與正式原始資料重複，容易造成程式讀錯資料。
